# Customer Churn Prediction System using AWS SageMaker

An end-to-end machine learning application designed to predict customer churn using historical customer behavior and subscription data. The project combines data analysis, machine learning, backend APIs, frontend integration, AWS cloud deployment, and automated retraining workflows.

## Project Status

Currently under development — building the ML pipeline, AWS deployment workflow, and automation components.
